FAQ

자주 묻는 질문

궁금한 점을 빠르게 확인하세요

코인 초보인데 사용할 수 있나요? +
네, API 키만 입력하면 자동으로 매매가 진행됩니다. 전문 지식이 필요 없으며, 설치부터 세팅까지 1:1로 도와드립니다.
내 자산이 안전한가요? +
API 키는 매매 권한만 허용합니다. 출금 기능은 비활성화되어 있어 누구도 자산을 인출할 수 없습니다. 자산은 항상 본인의 거래소 계좌에 보관됩니다.
손실이 나면 어떻게 되나요? +
모든 포지션에 자동 손절이 설정되어 있습니다. 1회 최대 손실을 제한하여 큰 손실을 방지합니다. 장기적으로 승률과 손익비로 수익을 만드는 구조입니다.
어떤 거래소를 지원하나요? +
현재 XT.com 거래소를 지원합니다. 향후 Bitget, Bybit 등 추가 거래소 연동을 준비 중입니다.
최소 투자금은 얼마인가요? +
최소 100 USDT부터 시작 가능합니다. 소액으로 먼저 테스트해보시는 것을 권장드립니다.
수익이 보장되나요? +
투자에 100% 수익 보장은 없습니다. 다만 검증된 전략과 철저한 리스크 관리로 장기적 수익 확률을 높이는 것이 RAID-X의 목표입니다.
